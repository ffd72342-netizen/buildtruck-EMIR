# BuildTruck EMIR

Modern, responsive, multi-page commercial landing website built as a **static site** using HTML, CSS, and vanilla JavaScript.

The project focuses on clean design, high conversion, fast loading, and easy deployment on any hosting.

## Features
- Multi-page structure
- Modern dark UI
- Mobile-first responsive layout
- SEO-friendly semantic HTML
- Clean and readable code
- No frameworks or build tools

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
